# Raw Data

This directory contains **unfiltered data** generated by the SentenceSmith pipeline before quality control.

---

## Files

| File | Source | Rows | Description |
|------|--------|------|-------------|
| `output_sentences_all.csv` | ChatGPT | 11,764 | Combined ChatGPT-sourced data (all packs merged) |
| `output_paws.csv` | PAWS | 9,995 | PAWS dataset processed through pipeline |
| `output_pack1.csv` | ChatGPT | 8,460 | ChatGPT batch 1/5 |
| `output_pack2.csv` | ChatGPT | 8,375 | ChatGPT batch 2/5 |
| `output_pack3.csv` | ChatGPT | 8,300 | ChatGPT batch 3/5 |
| `output_pack4.csv` | ChatGPT | 8,140 | ChatGPT batch 4/5 |
| `output_pack5.csv` | ChatGPT | 8,205 | ChatGPT batch 5/5 |

---

## Format

All files have 4 columns:
```
Original Sentence,Modified Sentence,Modification Type,semantic_relationship
```

**Note**: For quality-controlled evaluation data, see `../filtered_data/`
